# EcoTrackr

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Overview

EcoTrackr is an intuitive and user-friendly application designed to help individuals track their daily activities and estimate their carbon footprint. By providing actionable insights and personalized tips, EcoTrackr empowers users to make sustainable choices and contribute to a greener planet.

---

## Features

- Real-time carbon footprint calculation based on daily activities
- Personalized tips to reduce environmental impact
- Activity logging with categories like transportation, energy use, and diet
- Progress visualization with charts and statistics
- Goal setting to encourage sustainable habit formation
- Data export for personal records

---

## Tech Stack

- Frontend: React.js with Material-UI
- Backend: Node.js & Express
- Database: MongoDB Atlas
- Authentication: Firebase Auth
- Deployment: Vercel (frontend), Heroku (backend)

---

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/rachelnone556/EcoTrackr.git
   ```
2. Navigate to the project directory:
   ```
   cd EcoTrackr
   ```
3. Install backend dependencies:
   ```
   cd backend
   npm install
   ```
4. Install frontend dependencies:
   ```
   cd ../frontend
   npm install
   ```
5. Configure environment variables for backend (e.g., MongoDB URI, Firebase keys).
6. Run backend server:
   ```
   npm start
   ```
7. Run frontend development server:
   ```
   npm start
   ```

---

## Usage

- Sign up or log in using your email account.
- Log daily activities with estimated time and intensity.
- View your daily, weekly, and monthly carbon footprint reports.
- Explore personalized tips to reduce your footprint.
- Set goals and track your progress over time.
